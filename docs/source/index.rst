WOFOST Assumptions Tracking
---------------------------

This repository contains code examples for assumptions tracking of WOFOST Potato using behaviour driven development and model cards.

Communication
-------------

* `GitHub Discussions <https://github.com/PlantandFoodResearch/wofost-assumptions-tracking/discussions>`_ for questions.
* `GitHub issues <https://github.com/PlantandFoodResearch/wofost-assumptions-tracking/issues>`_ for bug reports and feature requests.
